# Overview

Este repositorio analiza bases nominales desde un enfoque estrictamente geográfico.

La prioridad es identificar:

- tablas con geografía utilizable
- nivel de codificación de sus campos
- capacidad de vincular domicilios con CUIL/CUIT
- fuentes oficiales para normalizar y codificar

## Salidas esperadas

- mapa claro de qué tablas sirven para geolocalización
- priorización de tablas para ETL geográfico
- criterios comunes de análisis entre esquemas
- base documental para normalización contra capas oficiales
