# Fuentes oficiales geográficas

Este documento reúne las fuentes oficiales y capas de referencia que se usarán para normalización, validación y codificación geográfica.

## Objetivos de estas fuentes

- normalizar provincia, departamento, municipio y localidad
- validar consistencia territorial
- codificar registros originalmente en texto libre
- servir de referencia maestra para joins geográficos
